    💡 Usage Tips

    • Add unwanted sites to the blacklist with one click. If you visit such a site, click the "Block This Site" button on the web extension popup.

    • It is not currently possible to edit blocking/redirecting rules. Simply delete the old rule and create a new one. Editing rules is planned to be added in future updates.
